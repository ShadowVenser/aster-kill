# ЛР 2 - ECS

Данный репозиторий содержит в себе примерный API для собственного ECS фреймворка. 
Для упрощения также добавлены комментарии, что и где необходимо дореализовать.
Само API не является единственно верным вариантом и может изменяться в процессе выполнения лабы в зависимости от ваших нужд и пожеланий.
